# UIUX reference (Getviews Studio)

Canonical **design + prototype source** for the Studio shell, Home, and related screens — replaces ad-hoc `src/make-import/` Figma Make dumps for parity work.

- **`shell.jsx`** — sidebar, top bar patterns  
- **`screens/*.jsx`** — screen-level JSX (read-only reference; production lives under `src/routes/`)  
- **`styles.css`** — token names align with `src/app.css` Phase A `--gv-*` (same palette intent as `:root` in this file)  
- **`app.jsx`**, **`data.js`**, **`icons.jsx`** — wiring and mock data for the standalone prototype  

Large raster screenshots are not stored here; keep the original zip in Downloads if needed.
